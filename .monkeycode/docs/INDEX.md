# 票夹管家 - 项目索引

## 项目概述

**项目名称：** 票夹管家  
**项目类型：** 个人票据收藏移动应用  
**核心功能：** 拍照存档证件、门票、车票、发票等各类票据，通过 OCR 自动提取信息并智能分类，方便日后查找与回忆。

### 设计原则

- **隐私优先：** OCR 识别由用户手动点击触发，绝不自动进行；数据默认本地存储。
- **智能 OCR：** 使用本地 RapidOCR（模型静默下载，精度优化）。
- **简洁界面：** 无底部导航，所有次要功能收于右上角折叠菜单。
